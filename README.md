# 🤖 VisionText AI Assistant

![AI Banner](https://images.unsplash.com/photo-1677442136019-21780ecad995)
---

## 🚀 Overview

**VisionText AI** is a simple and interactive web application built with **Streamlit** that uses AI to analyze images and generate intelligent descriptions.

Upload an image → get an AI-powered explanation instantly.

---

## ✨ Features

* 📷 Upload and preview images
* 🤖 AI-generated image descriptions
* ⚡ Fast and responsive interface
* ⚠️ Handles API errors (e.g. high demand)
* 🎨 Clean and user-friendly design

---

## 🛠️ Technologies Used

* **Python**
* **Streamlit**
* **Google Gemini API**

---

## ▶️ How to Run Locally

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the app

```bash
streamlit run app.py
```

---

## 📁 Project Structure

```
VisionText-AI/
│── app.py          # Main Streamlit interface  
│── chatbot.py      # AI processing logic  
│── requirements.txt  
│── .env            # API keys (not included publicly)  
```

---

## ⚠️ Important

Make sure you create a `.env` file with your API key:

```
GEMINI_API_KEY=your_api_key_here
```
